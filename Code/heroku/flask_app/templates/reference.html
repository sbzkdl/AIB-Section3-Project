Reference 페이지에서는 예측 모델를 돌렸을 때, 인기에 가장 영향을 많이 끼치는 4가지 특성만을 사용하여 레퍼런스 곡을 찾습니다. 인기있는 곡의 각 특성 수치는 다음과 같습니다.

Genre : Pop, Rock, Rap 장르가 인기도 Top 3입니다.
Danceability : 약 0.4 ~ 0.55의 지표와 0.7 이상의 지표를 가져야합니다.
Instrumentalness : 값이 상승할수록 인기도는 떨어집니다.
Loudness : 약 -17db 이상의 지표가져야 합니다.

레퍼런스 곡과 그 특성들을 참고하여 더 좋은 곡을 만들어주세요!
